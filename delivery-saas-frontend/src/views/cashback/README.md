# Módulo: Cashback (CASHBACK)

## Descrição
Views do módulo de cashback. Permite configurar percentual padrão, valor mínimo de resgate e regras por produto.

## Chave do módulo
`CASHBACK`

## Componentes
- **CashbackSettings.vue** — Configurações globais de cashback (habilitar/desabilitar, percentual, valor mínimo)

## Rotas
- `/settings/cashback`
