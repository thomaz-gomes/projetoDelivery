# Módulo: Cupons (COUPONS)

## Descrição
Views do módulo de cupons de desconto. Permite criar, listar e gerenciar cupons com regras de uso.

## Chave do módulo
`COUPONS`

## Componentes
- **CouponsList.vue** — Lista de cupons com filtros (ativo/inativo, tipo, afiliado)
- **CouponForm.vue** — Formulário de criação/edição de cupom

## Rotas
- `/coupons`, `/coupons/new`, `/coupons/:id/edit`
