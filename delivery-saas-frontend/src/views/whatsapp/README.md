# Módulo: WhatsApp (WHATSAPP)

## Descrição
Views do módulo de integração WhatsApp Business. Permite criar instâncias, conectar dispositivos via QR Code e gerenciar status.

## Chave do módulo
`WHATSAPP`

## Componentes
- **WhatsAppConnect.vue** — Gerenciamento de instâncias WhatsApp (criar, conectar QR, verificar status)

## Rotas
- `/settings/whatsapp`
