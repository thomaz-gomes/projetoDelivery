# Build resources

This directory holds platform-specific resources electron-builder picks up automatically.

Required for Windows installer:
- `icon.ico` — 256x256, ideally multi-resolution (16, 32, 48, 64, 128, 256).

Without `icon.ico`, electron-builder falls back to a generic Electron icon, but the build still succeeds. Drop the file here and rerun `npm run build:win`.

Required for macOS (when D2 is enabled):
- `icon.icns` — 1024x1024 root, packed via `iconutil`.

Required for auto-update (D3):
- `latest.yml` / `latest-mac.yml` are generated by electron-builder during `npm run dist` — don't commit them.
