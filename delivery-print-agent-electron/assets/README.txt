Coloque os seguintes arquivos nesta pasta:

  icon.ico       - Ícone do aplicativo (256x256 ICO)
  tray-green.png - Ícone da bandeja: conectado  (16x16 PNG)
  tray-red.png   - Ícone da bandeja: desconectado (16x16 PNG)
  tray-yellow.png - Ícone da bandeja: conectando (16x16 PNG)

Sugestão: use ferramentas como https://realfavicongenerator.net/ para gerar os ícones.

Se os arquivos de bandeja não existirem, o agente funciona normalmente,
apenas sem ícone colorido na System Tray.
