# Gerenciamento de Usuarios

## O que e
O gerenciamento de usuarios permite criar e administrar as contas de acesso ao painel da loja. Cada usuario tem um perfil (role) que define o que ele pode fazer no sistema.

## Como acessar
Menu lateral → Configuracoes → Usuarios

## Como usar

### Criar um usuario
1. Clique em **Novo usuario**
2. Preencha o **Nome** e **Email**
3. Opcionalmente, informe o **WhatsApp**
4. Selecione o **Perfil (Role)**:
   - **Admin**: acesso completo ao painel
   - **Atendimento**: acesso limitado para operacao do dia a dia
5. Defina uma **Senha** (opcional)
6. Clique em **Salvar**

### Editar um usuario
1. Clique em **Editar** ao lado do usuario
2. Altere os dados necessarios
3. Para alterar a senha, preencha o campo de senha
4. Clique em **Salvar**

### Remover um usuario
Clique em **Remover** ao lado do usuario. Apenas administradores com nivel superior podem remover usuarios.

### Configurar permissoes por perfil
1. Clique em **Permissoes por role**
2. Selecione o perfil desejado na lista da esquerda
3. Marque ou desmarque as permissoes disponiveis
4. Clique em **Salvar**

## Duvidas frequentes

**Quais sao os perfis disponiveis?**
Admin (acesso completo) e Atendimento (acesso limitado para operacoes do dia a dia como receber pedidos).

**Posso personalizar as permissoes de cada perfil?**
Sim. Use o botao "Permissoes por role" para definir exatamente o que cada perfil pode acessar.

**Como redefinir a senha de um usuario?**
Edite o usuario e preencha o campo de senha com a nova senha desejada.

**Posso pesquisar usuarios?**
Sim. Use o campo de pesquisa para buscar por nome ou email.
