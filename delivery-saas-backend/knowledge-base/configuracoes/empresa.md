# Configuracoes da Loja

## O que e
As configuracoes da loja permitem definir os dados basicos do seu estabelecimento, como nome, endereco, telefone, CNPJ e dados fiscais. Cada loja pode ter suas proprias configuracoes.

## Como acessar
Menu lateral → Configuracoes → Lojas → clique em Editar na loja desejada

## Como usar

### Aba Geral
1. Preencha o **Nome** da loja
2. Ative ou desative a loja com o botao **Loja ativa**
3. Informe o **Endereco** completo
4. Selecione o **Estado** e a **Cidade**
5. Preencha o **Telefone** e **WhatsApp**
6. Informe o **CNPJ** da loja

### Aba Fiscal
Disponivel para lojas com emissao de nota fiscal configurada:
1. Preencha a **Inscricao Estadual (IE)**
2. Selecione o **Timezone** correto
3. Informe a **Razao Social**
4. Complete os dados de endereco do emitente

### Aba Precificacao
Configuracoes relacionadas a formacao de precos dos produtos.

### Pausar a loja temporariamente
Use o botao de loja ativa para desativar temporariamente. Uma pausa programada aparecera com o tempo restante e a opcao de cancelar.

## Duvidas frequentes

**Posso ter mais de uma loja?**
Sim. Acesse Configuracoes → Lojas para criar novas lojas, de acordo com o limite do seu plano.

**Preciso preencher os dados fiscais?**
Apenas se voce for emitir nota fiscal eletronica (NF-e/NFC-e) pelo sistema.

**O que acontece se eu desativar a loja?**
A loja ficara indisponivel para novos pedidos ate ser reativada.
