# Pedido Manual

## O que e
O pedido manual permite que voce crie um pedido diretamente pelo sistema, sem que o cliente precise fazer pelo cardapio online. E util para pedidos recebidos por telefone, WhatsApp ou no balcao.

## Como acessar
Menu lateral → Pedidos → Botao "Novo pedido" (no topo do painel)

## Como usar
1. No painel de pedidos, digite o **telefone do cliente** no campo de busca e clique para iniciar um novo pedido.
2. O assistente de pedido (PDV) sera aberto.
3. Selecione os **produtos** do cardapio que o cliente deseja.
4. Escolha os **complementos** de cada produto, se houver.
5. Informe os dados do cliente (nome, endereco de entrega, etc.).
6. Selecione a **forma de pagamento**.
7. Confira o resumo do pedido e clique em **Finalizar**.
8. O pedido sera criado e aparecera no painel de pedidos automaticamente.

## Duvidas frequentes

**O pedido manual aparece no painel como os outros?**
Sim. Apos criado, ele aparece na coluna "Novos / Em preparo" e segue o mesmo fluxo de status dos demais pedidos.

**Posso criar pedido manual para retirada no balcao?**
Sim. Na hora de criar o pedido, voce pode escolher se e entrega ou retirada.

**O cliente recebe alguma notificacao?**
Se o cliente tiver WhatsApp cadastrado e o modulo de notificacoes estiver ativo, ele pode receber atualizacoes automaticas sobre o pedido.
