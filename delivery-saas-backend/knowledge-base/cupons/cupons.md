# Cupons de Desconto

## O que e
Cupons de desconto permitem oferecer descontos aos seus clientes por meio de codigos promocionais. Voce pode criar cupons com desconto em percentual ou valor fixo, definir limites de uso e associar cupons a afiliados.

## Como acessar
Menu lateral → Cupons

## Como usar

### Criar um cupom
1. Na lista de cupons, clique em **Novo Cupom**
2. Preencha o **Codigo** do cupom (ex: PROMO10)
3. Adicione uma **Descricao** opcional para identificar o cupom
4. Escolha o **Tipo**: Percentual ou Valor absoluto
5. Informe o **Valor** do desconto. Para percentual, use valores entre 0 e 1 (ex: 0.10 = 10%)
6. Opcionalmente, vincule o cupom a um **Afiliado**
7. Defina o **Status** como Ativo ou Inativo
8. Se desejar, configure uma **Data de expiracao**, **Subtotal minimo**, **Maximo de usos global** e **Maximo de usos por cliente**
9. Clique em **Salvar**

### Gerenciar cupons
- Use os filtros na lista para buscar por status (Ativo/Inativo), tipo (Percentual/Valor) ou afiliado
- Use a barra de pesquisa para buscar pelo codigo do cupom
- Clique no icone de lapis para editar um cupom existente
- Clique no icone de lixeira para excluir um cupom

## Duvidas frequentes

**Como funciona o desconto percentual?**
O valor deve ser informado como decimal. Por exemplo, 0.10 significa 10% de desconto, 0.25 significa 25%.

**Posso limitar quantas vezes um cupom pode ser usado?**
Sim. Use os campos "Maximo de usos" (limite global) e "Maximo de usos por cliente" (limite por pessoa).

**O que acontece quando um cupom expira?**
O cupom nao podera mais ser utilizado pelos clientes apos a data de expiracao.

**Posso exigir um valor minimo de pedido para usar o cupom?**
Sim. Preencha o campo "Subtotal minimo" com o valor desejado.
