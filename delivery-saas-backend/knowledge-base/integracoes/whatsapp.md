# WhatsApp - Conexao e Inbox

## O que e
A integracao com WhatsApp permite conectar um numero de telefone ao sistema para receber e enviar mensagens diretamente pelo painel. Voce pode criar varias instancias (uma para cada numero ou loja) e gerenciar todas as conversas em um unico lugar atraves do Inbox.

## Como acessar
- Conexao: Menu lateral → Integracoes → WhatsApp
- Inbox (mensagens): Menu lateral → Inbox

## Como usar

### Conectar o WhatsApp

**Criar uma nova instancia**
1. Clique em **Adicionar nova instancia**.
2. Informe o **Nome da instancia** (exemplo: "loja-01").
3. Defina um **Nome exibido** (opcional).
4. Selecione as **Lojas** que essa instancia vai atender ou marque "Atribuir a todas as lojas".
5. Clique em **Criar e atribuir**.

**Escanear o QR Code**
1. Na lista de instancias, clique em **Gerenciar**.
2. Abra o WhatsApp no celular.
3. Va em **Configuracoes** → **Dispositivos conectados** → **Conectar dispositivo**.
4. Escaneie o QR code exibido na tela.
5. Aguarde o status mudar para "Conectado".

### Gerenciar instancias
- **Gerenciar**: abre o painel com QR code, status e opcoes de configuracao.
- **Atribuir/Reconfigurar**: muda quais lojas sao atendidas por essa instancia.
- **Remover**: exclui a instancia permanentemente.

### Ativar notificacoes
Use o botao **Ativar notificacao** no topo da pagina para que o sistema envie mensagens automaticas aos clientes sobre o status dos pedidos via WhatsApp.

### Usar o Inbox (central de mensagens)
1. Acesse Menu lateral → Inbox.
2. No lado esquerdo, veja a lista de conversas.
3. Clique em uma conversa para abrir o chat.
4. Digite sua mensagem e envie.
5. No lado direito, veja os dados do contato.
6. Novas mensagens recebidas emitem um alerta sonoro e uma notificacao no navegador.

## Duvidas frequentes

**Posso usar o mesmo numero no celular e no sistema?**
Sim. O WhatsApp permite varios dispositivos conectados ao mesmo tempo.

**O que acontece se o celular ficar sem internet?**
O sistema continua funcionando normalmente por um tempo. Porem, se o celular ficar muito tempo desconectado, pode ser necessario escanear o QR code novamente.

**Posso ter mais de um numero de WhatsApp?**
Sim. Crie uma instancia para cada numero. Cada instancia pode ser atribuida a lojas diferentes.

**As mensagens sao salvas?**
Sim. Todo o historico de conversas fica salvo no sistema e pode ser consultado a qualquer momento pelo Inbox.
